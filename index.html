🔒 What’s New in Cisco FMC 7.6+ — Security Features

Loading latest from cisco.com…

Live security-feature highlights from Cisco Secure Firewall Management Center release notes (7.6.0 → 7.6.5). Suggested release: FMC 7.6.5 ⭐  /  FTD 7.6.4 ⭐.

FMC New Features by Release ↗ FTD 7.6.x Release Notes ↗ What’s New in 7.6 (Cisco) ↗
7.6.0 SnortML — ML-based exploit detection

New Snort 3 inspector snort_ml uses neural-network ML to detect known and 0-day attacks (currently SQL injections via HTTP URI) without writing per-CVE rules. Disabled by default except in Maximum Detection policy.

7.6.0 QUIC Decryption

Decryption Policy can now apply to QUIC sessions, enabling full IPS / malware inspection of encrypted HTTP/3 traffic. Off by default; enable per decryption rule.

7.6.0 EVE Exception List

Bypass Encrypted Visibility Engine block verdicts for trusted destination networks or process names. Bypassed connections are tagged EVE Exempted in events.

7.6.0 Simplified Do-Not-Decrypt Wizard

Multistep decryption-policy wizard lets admins define outbound decryption exclusions without disturbing existing decryption rules.

7.6.0 MITRE ATT&CK Enrichment in Unified Events

Single-pane Unified Event Viewer with MITRE technique tagging, attack progression graph, and EVE/IOC contextual enrichment.

7.6.0 AI Assistant for FMC

Cloud-delivered AI Assistant integrated into FMC for natural-language policy queries, configuration insight, and optimization guidance.

7.6.0 Policy Analyzer & Optimizer

Cloud-delivered Access Control policy review with actionable recommendations — detects shadowed, redundant, and overly broad rules.

7.6.0 Snort 3 CPU & Rule Profilers

New profilers expose per-rule and per-module CPU cost so analysts can pinpoint expensive Snort 3 rules without lab packet captures.

7.6.0 RADIUS Message-Authenticator Enforcement

Mitigates CVE-2024-3596 (Blast-RADIUS): require Message-Authenticator attribute in all RADIUS responses for VPN, AnyConnect, and admin auth.

7.6.1 Migrate Firepower 4100/9300 → Secure Firewall 3100/4200

Guided migration path to retire Firepower 4100/9300 chassis onto modern Secure Firewall hardware while preserving policy.

7.6.4 FMCv High Availability on OpenStack

FMC Virtual now supports HA pairing on OpenStack — closing a long-standing parity gap for private-cloud deployments.

7.6.5 Suggested Release — FMC 7.6.5 ⭐

Cisco-recommended FMC release pairing with FTD 7.6.4. Bug fixes and stability hardening on top of the 7.6 security feature set above. See New Features by Release for the full bug list.
